# CP009智能电表

基于STM32F103C8T6单片机的智能电表系统，支持WiFi远程控制、温湿度监测、功率监测和定时控制功能。

## 项目简介

本项目是一款面向毕设的智能电表作品，采用STM32F103C8T6作为主控芯片，通过ESP8266模块实现WiFi通信，支持远程监控和控制。

## 硬件架构

| 模块 | 型号 | 功能 |
|------|------|------|
| 主控芯片 | STM32F103C8T6 | 系统核心控制 |
| WiFi模块 | ESP8266 | 无线通信（AP模式） |
| 温湿度传感器 | DHT11 | 环境温湿度监测 |
| 实时时钟 | DS1302 | 时间管理 |
| 显示模块 | OLED 128x64 | 状态显示 |
| 语音模块 | JQ8900 | 报警提示 |
| 电源监测 | 串口通讯模块 | 电压/电流/功率监测 |

## 主要功能

### 1. 电源监测
- 实时监测电压、电流、功率
- 通过串口通讯读取传感器数据
- 数据格式：temp:%d,humi:%d,U:%.1f,I:%.4f,P:%.1f,

### 2. 温湿度监测
- DHT11传感器实时采集环境温湿度
- 温度监测范围：0-50°C
- 湿度监测范围：20-90%RH

### 3. WiFi远程控制
- ESP8266工作在AP模式
- WiFi名称：GDP001
- WiFi密码：12345678
- TCP服务器端口：8080

### 4. 定时控制
- 支持定时开关电源
- 定时精度：秒级
- 可设置小时、分钟、秒

### 5. 阈值保护
- 温度阈值报警（默认40°C）
- 湿度阈值报警（默认80%）
- 超阈值自动切断电源并语音提醒

## 控制命令

通过TCP连接发送命令到ESP8266（192.168.4.1:8080）：

| 命令格式 | 功能说明 | 示例 |
|----------|----------|------|
| mode=X | 切换模式（0自动/1手动） | mode=1 |
| temp_max=X | 设置温度阈值 | temp_max=35 |
| humi_max=X | 设置湿度阈值 | humi_max=75 |
| hour=X | 设置定时小时 | hour=10 |
| minute=X | 设置定时分钟 | minute=30 |
| sec=X | 设置定时秒 | sec=0 |
| power_state=X | 控制电源（0关/1开） | power_state=1 |

### 组合命令示例

```bash
# 切换到手动模式并开启电源
mode=1,power_state=1

# 设置温度阈值35°C，湿度阈值75%
mode=1,temp_max=35,humi_max=75

# 设置10分钟后执行
mode=1,hour=0,minute=10,sec=0
```

## 使用方法

### 1. 硬件连接

按照原理图连接各模块：
- ESP8266连接到USART1
- DHT11连接到GPIO
- DS1302连接到GPIO
- OLED连接到I2C接口
- JQ8900连接到USART3
- 电源监测模块连接到USART2

### 2. 编译烧录

1. 使用Keil打开 project.uvprojx 工程文件
2. 编译项目：Project → Build Target
3. 烧录到单片机：Flash → Download

### 3. WiFi连接

1. 单片机启动后，ESP8266会创建WiFi热点
2. 使用手机或电脑连接热点"GDP001"
3. 密码：12345678
4. 建立TCP连接到 192.168.4.1:8080

### 4. 远程控制

#### 命令行工具

```bash
python tcp_client.py
```

#### Web控制页面

在浏览器中打开 control.html 文件，即可看到可视化控制界面。

## 技术亮点

1. 多传感器数据融合：整合温湿度、电压、电流、功率等多种数据
2. 远程控制：通过WiFi实现远程监控和控制
3. 实时时钟：DS1302提供精确时间管理
4. 智能保护：超阈值自动切断电源并语音报警
5. OLED显示：实时显示系统状态

## 注意事项

1. ESP8266需要单独供电或确保供电充足
2. WiFi热点仅供测试使用，不建议长期使用默认密码
3. 电源监测模块使用Modbus协议
4. 确保各模块电平匹配（3.3V系统）
